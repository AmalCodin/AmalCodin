## 🚀 What I'm Building Right Now

### 🛍️ Svetlana Flowers — Full-Stack E-Commerce Platform
A production e-commerce site for a live flower brand.
- **Next.js 14 (App Router)** + **Supabase** + **Stripe** + **Zustand**
- 1,355+ products seeded from the Udora supplier API across two shops
- Cart, checkout, and Stripe payment flows built end-to-end
- Live search with suggestions, delivery location picker, admin auth
- Solved real production bugs: async params in Next.js 15+, Supabase key migrations, and a tricky input-focus-loss issue from component re-mounting

### 🌹 Multi-Brand Order & Catalog Management Suite
Internal tooling for running three flower brands at once.
- **FastAPI** backend + HTML/JS dashboard for live order management across stores
- Deep integration with the **Udora/Flowwow Seller API**, including pagination, discount logic, and cross-verification against the supplier's own dashboard
- A standalone **Catalog Manager** (async `httpx`, six-shop token map) for bulk product control
- An **18-slot shelf rotation system** (hero, seasonal, occasion, AOV, new, anchor slot types) with weekly rotation logic to optimize storefront merchandising
- Drafted a formal API feedback report for the Udora platform team

### 🤖 WhatsApp Ordering Bot — Bloom Roses
Conversational commerce, no app required.
- **n8n** workflow engine + **Google Gemini 2.0 Flash** + **WhatsApp Business Cloud API**
- **Supabase/PostgreSQL** backend for orders and conversation state
- Iterated through multiple workflow versions to harden the conversation logic

### 📈 AI Gold Trading Bot — XAUUSD on MetaTrader 5
Applying ML to my own trading.
- Multi-timeframe **XGBoost** model (M5/M15/M30/H1, 63 engineered features)
- **Streamlit** dashboard with full CSV trade logging
- Refined exit-score logic, MTF trend alignment, and confidence thresholds through extensive backtesting
